# Buz-Houses
> Este proyecto fue creado para aprender las nuevas herramientas y utilidades que fuimos aprendiendo durante la cursada

> Como mi tercera pre-entega dieseñé de mejor manera todo mediante scss solamente modificando algunas cosas de gusto estetico. Se basa en una empresa de compra, renta y venta de propiedades en la localidad de Buzios, Brasil

----

## Herramientas utilizadas

- CSS
- HTML
- SCSS  como pre-procesador
